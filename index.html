JavaScript Programming Cheat Sheet

1. Hello, World
Basic syntax to print "Hello, World!" to the console.
console.log("Hello, World!");
2. Data Types
Common JavaScript data types.
let num = 42; // Number
let str = "Hello"; // String
let isTrue = true; // Boolean
let obj = { name: "John", age: 30 }; // Object
let arr = [1, 2, 3]; // Array
3. Variables
Different ways to declare variables in JavaScript.
let x = 10;
const y = 20;
var z = 30;
4. Operators
Basic arithmetic and comparison operators.
let sum = 5 + 3; // Addition
let diff = 5 - 3; // Subtraction
let product = 5 * 3; // Multiplication
let quotient = 5 / 3; // Division
5. Loops
Basic loop structures in JavaScript.
for (let i = 0; i < 5; i++) {
    console.log(i);
}

while (x < 5) {
    console.log(x);
    x++;
}
6. Control Flow
Conditional statements and branching logic.
if (x > 10) {
    console.log("Greater");
} else {
    console.log("Smaller");
}

switch (color) {
    case "red":
       console.log("Red");
    break;
}
7. Functions
Defining and calling functions.
function greet(name) {
    return "Hello, " + name;
}
let greeting = greet("John");
console.log(greeting);
8AS(1.) Destructuring
Extracting values from arrays or objects into variables.
let [a, b] = [1, 2];
let {name, age} = {name: "John", age: 30};
8AS(2.) Spread & Rest Operators
Spread copies elements from arrays or objects, rest collects multiple elements into one.
let arr1 = [1, 2, 3];
let arr2 = [...arr1, 4]; // Spread
let sum = (...args) => args.reduce(function (acc, num) {
    return acc + num;
});
8AS(3.) Template Literals
Using backticks for multi-line strings and string interpolation.
let name = "John";
let greeting = `Hello, ${name}`;
8AS(4.) Optional Chaining
Access properties of an object without worrying if itâ€™s undefined or null.
let user = { profile: { name: "John" } };
let username = user?.profile?.name;
8AS(5.) Modules
Import and export functions and variables between JavaScript files.
import { myFunction } from './myModule';
export function myFunction() { ... };
9. Closures
A function that retains access to its lexical scope even after the function is executed.
function outer() {
    let outerVar = 10;
    return function inner() {
       console.log(outerVar);
    };
}
let innerFunc = outer();
innerFunc();
10. Higher-order Functions
Functions that take other functions as arguments or return them.
let double = function(x) {
    return x * 2;
};
let result = [1, 2, 3].map(double);
11. The this Keyword
Refers to the object that the function is called upon.
let person = {
    name: "John",
    greet: function() {
       console.log(`Hello, ${this.name}`);
    }
};
person.greet();
12. Event Delegation
A method to handle events efficiently on dynamically created elements.
document.querySelector('ul').addEventListener('click', function(event) {
    if (event.target.tagName === 'LI') {
       console.log('List item clicked');
    }
});
13. Async Iteration (for-await-of)
Iterate over asynchronous data sources like streams or async generators.
async function fetchData() {
    for (await let data of getDataAsync()) {
       console.log(data);
    }
}
fetchData();
14. Map & Set
Maps store key-value pairs, and Sets store unique values.
let map = new Map();
map.set("name", "John");
let set = new Set();
set.add(1);
set.add(2);
15. Functional Programming
Using pure functions, immutability, and higher-order functions.
let add = (a, b) => a + b;
let numbers = [1, 2, 3];
let sum = numbers.reduce(add, 0);
16. Debouncing & Throttling
Control the frequency of function executions to optimize performance.
// Debounce
function debounce(func, delay) {
    let timer;
    return function(...args) {
       clearTimeout(timer);
       timer = setTimeout(function() {
           func(...args);
       }, delay);
};
}
17. Prototypes & Inheritance
JavaScript objects can inherit properties from other objects through prototypes.
function Person(name) {
    this.name = name;
}
Person.prototype.sayHello = function() {
    console.log(`Hello, ${this.name}`);
};
let person = new Person("John");
person.sayHello();
18. Memory Management
Understanding garbage collection and memory leaks in JavaScript.
// Avoid memory leaks by clearing timers and event listeners
let timer = setInterval(function() { ... }, 1000);
clearInterval(timer);
19. JavaScript Design Patterns
Common design patterns used in JavaScript for structuring code.
let singleton = (function() {
    let instance;
    return function() {
       if (!instance) {
           instance = new Object();
       }
       return instance;
    };
})();
20. Web APIs
JavaScript APIs for interacting with the browser environment.
fetch("https://api.example.com").then(response => response.json()).then(data => {
    console.log(data);
});
21. DOM Manipulation
Interacting with HTML elements using JavaScript.
document.querySelector('#myElement').textContent = "New Text";
document.querySelector('#myButton').addEventListener('click', () => {
    alert("Button clicked!");
});
22. Promises & Async/Await
Handling asynchronous code with Promises and Async/Await.
function fetchData() {
    return new Promise((resolve, reject) => {
       resolve("Data");
    });
}
async function asyncFetch() {
    let result = await fetchData();
    console.log(result);
}
asyncFetch();
23. Array/Object Methods
Commonly used methods for arrays and objects.
let arr = [1, 2, 3];
arr.map(x => x * 2);
let obj = {
    name: "John",
    age: 30
};
Object.keys(obj);
Object.values(obj);
24. Class Syntax
Modern way to define classes and objects in JavaScript.
class Person {
    constructor(name, age) {
       this.name = name;
       this.age = age;
    }
}
let person = new Person("John", 30);
25. Event Loop & Async Concepts
Understanding the event loop, call stack, and asynchronous behavior in JavaScript.
console.log("Start");
setTimeout(() => {
    console.log("Timeout");
}, 1000);
console.log("End");
26. Error Handling
Using try-catch-finally to handle errors in JavaScript.
try {
    let result = JSON.parse("Invalid JSON");
} catch (e) {
    console.log("Error: " + e.message);
}
finally {
    console.log("Cleanup");
}
